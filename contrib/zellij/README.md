# Lavi for Zellij

<a href="https://github.com/zellij-org/zellij">Zellij</a>: Terminal workspace and multiplexer

## Installation

1. Copy [`lavi.kdl`](./lavi.kdl) to `~/.config/zellij/themes/lavi.kdl`
2. Set `theme "lavi"` in your `~/.config/zellij/config.kdl`

---

Part of [Lavi](https://github.com/b0o/lavi), a soft and sweet colorscheme.
