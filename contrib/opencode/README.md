# Lavi for Opencode

![Opencode screenshot](https://github.com/user-attachments/assets/03d3a17c-310f-44fe-b554-b4ab6dfead8d)

<a href="https://github.com/opencode-ai/opencode">Opencode</a>: TUI for coding with LLMs from the terminal

## Installation

1. Copy [`contrib/opencode/lavi.json`](./lavi.json) to `~/.config/opencode/themes/lavi.json`
2. Set `{ "theme": "lavi" }` in your `~/.config/opencode/opencode.jsonc` or select it from the UI theme picker

---

Part of [Lavi](https://github.com/b0o/lavi), a soft and sweet colorscheme.
